QA Automation with PyTest

This repository demonstrates basic QA automation skills using Python and PyTest.

Features
- Unit testing (calculator functions)
- API testing using REST endpoints
- Test case design and validation
- Error handling and edge case testing

Tech Stack
- Python
- PyTest
- Requests (API testing)

How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   pytest -v

Learning Outcome
- Writing automated test cases using PyTest
- API response validation
- Handling edge cases and exceptions
- Understanding QA automation basics

Future Improvements
